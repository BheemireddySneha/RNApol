# RNApol
codes used in RNA polymerase project
# Normal mode analysis
1. anm_rbpA.py -> performs normal mode analysis and outputs Z-Score
2. Z_score_plot.py -> plotting the z-score
3. dccm.ipynb -> perform dccm and outputs cross-correlation plot
4. prs.py -> performs perturbation response scanning
5. MD_PRS.ipynb -> performs PRS for MD data
6. CA_RMSD_plot.ipynb -> C-alpha RMSD plotting

# mmpbsa
1. mmpbsa.in -> input for mmpbsa calculations
